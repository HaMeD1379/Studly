## Running Biome Lint
- Type "cd ./apps/frontend"
- Type "npm ci"
- Type "npm run lint"
If there are things to fix, you can then do:
- Type "npm run lint:fix"

## Running Biome Format
- Type "cd ./apps/frontend"
- Type "npm ci"
- Type "npm run format"
If there are things to fix, you can then do:
- Type "npm run format:fix"

## Running Biome with Fix on all
- Type "cd ./apps/frontend"
- Type "npm ci"
- Type "npm run biome:fix"
